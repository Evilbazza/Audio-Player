STEP 1 — Create a Google Cloud Project

  • Go to https://console.cloud.google.com and sign in
  • Click Select a project at the top → New Project
  • Give it any name (e.g. Audio Player), click Create
  • Wait for it to create, then confirm it is selected in the top bar

STEP 2 — Enable the Google Drive API

  • Left menu → APIs & Services → Library
  • Search for Google Drive API, click it, then click Enable

STEP 3 — Configure the OAuth Consent Screen

  • Left menu → APIs & Services → OAuth consent screen
  • Select External, click Create
  • Fill in App name, User support email, Developer contact email
  • Click Save and Continue through all remaining steps
  • On the final summary page, click Back to Dashboard
  • Click Publish App and confirm — this allows your Google account to sign in

STEP 4 — Create an OAuth Client ID

  • Left menu → APIs & Services → Credentials
  • Click + Create Credentials → OAuth client ID
  • Application type: Web application
  • Under Authorised JavaScript origins → + Add URI
  • Type exactly: null — required for local HTML files in Android Chrome
  • Click Create — copy the Client ID from the popup
  • It looks like: 123456789-abc123.apps.googleusercontent.com

STEP 5 — Use this App

  • Paste your Client ID into the field below
  • Tap Sign in with Google and approve access
  • Open a folder in Google Drive — copy the long ID at the end of the URL after /folders/
  • Paste that Folder ID and tap Load & Play
ℹ After completing setup once, you can collapse this panel — your credentials are never stored.
